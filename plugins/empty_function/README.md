# Плагин: Проверка на пустые функции

Этот плагин находит функции и методы, тело которых состоит исключительно из `pass`, оператора `...` (Ellipsis) или только строки-документации (docstring).

Такие пустые конструкции часто остаются после работы языковых моделей, когда они генерируют "заглушки" (stubs) для будущего кода, но не реализуют саму логику.

Плагин автоматически игнорирует методы, отмеченные декоратором `@abstractmethod`, так как для абстрактных методов пустое тело является нормальной практикой.
